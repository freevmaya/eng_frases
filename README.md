# English Phrases Trainer

Приложение для изучения английских фраз с озвучкой и настройками.

## Особенности

1. **Три списка фраз**:
   - Past Simple (активный залог)
   - Past Simple (пассивный залог)
   - Future Simple (пассивный залог)

2. **Режимы изучения**:
   - Русский → Английский
   - Английский → Русский
   - Оба направления (два варианта)

3. **Настройки**:
   - Скорость речи
   - Паузы между фразами и языками
   - Порядок фраз (последовательный/случайный)
   - Выбор списка фраз

4. **Озвучка**:
   - Использует Web Speech API
   - Поддержка русского и английского языков
   - Регулировка скорости

## Установка

Просто загрузите все файлы на веб-сервер с поддержкой PHP.

## Использование

1. Откройте `index.php` в браузере
2. Нажмите кнопку настроек (иконка шестерёнки)
3. Выберите желаемые параметры
4. Нажмите "Старт" для начала обучения

## Технологии

- HTML5, CSS3, JavaScript
- Bootstrap 5 для интерфейса
- jQuery для упрощения DOM-манипуляций
- Web Speech API для синтеза речи
- PHP для конфигурации

## Требования к браузеру

- Современный браузер с поддержкой Web Speech API
- Chrome, Firefox, Edge последних версий
